# Cloudflare Redirect Rules 설정 가이드 (수동)

## 🤖 AI 응답
Cloudflare 대시보드에서 접속자의 브라우저 언어에 따라 자동으로 해당 언어 페이지(예: 한국어 사용자는 `/ko/`로)로 이동시켜주는 **리다이렉트 규칙(Redirect Rules)**을 설정하는 방법을 아주 쉽고 자세하게 단계별로 설명해 드릴게요.

### 🛠️ 준비 사항
먼저 브라우저를 열고 [Cloudflare 대시보드(dash.cloudflare.com)](https://dash.cloudflare.com/)에 로그인해 주세요.

### 1단계: 설정 메뉴 찾아가기
1. Cloudflare에 로그인한 후, 메인 화면에서 내 사이트인 **puzzlevault.pages.dev** (또는 연결해 둔 커스텀 도메인)를 클릭하세요.
2. 화면 왼쪽에 있는 사이드바 메뉴들을 쭉 내려서 **[Rules] (규칙)** 메뉴를 찾아 클릭하세요.
3. 하위 메뉴가 열리면, 그중에서 **[Redirect Rules] (리다이렉트 규칙)**을 클릭합니다.
4. 화면 오른쪽에서 파란색 **[Create rule] (규칙 생성)** 버튼을 클릭하세요.

### 2단계: 한국어(ko) 자동 이동 규칙 만들기
1. **Rule name (규칙 이름)**: `한국어 자동 이동` 이라고 알아보기 쉽게 적어주세요.
2. 그 아래 **When incoming requests match... (수신 요청이 일치하는 경우)** 섹션에서 `Custom filter expression (사용자 지정 필터 표현식)`이 선택되어 있는지 확인하세요.
3. 이제 조건을 입력하는 곳에 다음 코드를 복사해서 붙여넣으세요. 파란색 글씨인 **Edit expression**을 클릭하고 네모난 텍스트 창에 아래 코드를 넣으면 편합니다.
   ```text
   (http.request.accepted_languages[0] eq "ko" and http.request.uri.path eq "/")
   ```
4. **Then... (그럼 어떻게 할 것인가)** 섹션으로 내려옵니다.
   - **Type (유형)**: `Dynamic (동적)`을 선택하세요.
   - **Status code (상태 코드)**: `302`를 선택하세요. (임시 이동이라는 뜻입니다)
   - **URL**: `concat("https://puzzlevault.pages.dev/ko/", http.request.uri.query)` 또는 `https://puzzlevault.pages.dev/ko/` 를 적어주세요.
   - **Preserve query string (쿼리 문자열 보존)**: 만약 체크박스가 보인다면 무조건 **체크(Yes)**해 주세요.
5. 맨 아래 파란색 **[Deploy] (배포)** 버튼을 누르면 첫 번째 한국어 규칙이 완성됩니다!

### 1. [Rules] 메뉴를 찾는 방법 (개인 도메인이 있는 경우)
만약 `.com`, `.net`, `.io` 같은 개인 도메인(예: `puzzlevault.com`)을 Cloudflare에 연결해 두셨다면 아래 방법으로 메뉴를 찾을 수 있습니다.

- 이후 왼쪽 사이드바에서 **[Rules]** 메뉴를 찾을 수 있습니다.

### 2. 개인 도메인 없이 `pages.dev` 기본 주소만 쓰는 경우

**💡 대안 가이드 (어어떻게 해야 하나요?)**
하지만 걱정하지 않으셔도 됩니다!

## 🤖 AI 응답
사용자님의 예리한 지적대로, **Cloudflare Pages Functions** 미들웨어를 활용하는 방식이 SEO 향상과 화면 깜빡임 방지 두 마리 토끼를 모두 잡을 수 있는 훨씬 더 우수하고 진보된 방법입니다!

제안해주신 아이디어를 적극 수용하여 다음과 같이 아키텍처를 업그레이드 완료했습니다.

1. **`functions/_middleware.js` 서버리스 라우팅 생성 (완료)**
   루트 경로(`/`)로 들어오는 요청을 가로채서(Intercept), 방문자의 `Accept-Language` 헤더와 명시적 언어 설정 쿠키(`pv_lang`)를 검사한 후 적절한 언어 디렉토리(예: `/ko/`)로 `HTTP 302 동적 리다이렉트`를 수행하는 미들웨어 스크립트를 작성했습니다. 
   - 이 코드는 브라우저가 사용자 화면에 아무것도 그리기 전 Cloudflare 엣지 노드 단에서 실행되므로 **단 0.1초의 렌더링 깜빡임도 완전히 사라지며**, 
   - 구글 검색 봇이 수집할 때에도 서버 차원의 HTTP 리다이렉트를 명확히 따르게 되어 **가장 강력한 다국어 SEO 효과**를 발휘합니다.

2. **`js/i18n.js` 쿠키 동기화 로직 추가 (완료)**
   사용자가 언어 선택기 모양을 클릭해 수동으로 언어를 변경할 경우 기존에는 `localStorage`에만 언어 코드가 저장되었습니다. `Cloudflare Functions`는 보안상 사용자의 로컬스토리지를 읽어올 수 없으므로, 언어 선택 시나 초기 렌더링 시 **`document.cookie` (유효기간 1년)에도 동일하게 언어 정보(`pv_lang`)를 굽도록 패치**했습니다.

이제 수동 대시보드 조작 없이, Cloudflare Pages로 코드를 배포하시기만 하면 자동으로 깜빡임 없는 가장 완벽한 형태의 다국어 전환 시스템이 작동합니다. 이 내용 역시 `walkthrough.md` 최하단에 상세히 기록해 두었습니다!
