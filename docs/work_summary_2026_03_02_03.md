# PuzzleVault 작업 내역 요약 (어제 ~ 오늘)
**작성일시**: 2026년 3월 3일

본 문서는 최근 양일간(최근 9개의 대화 세션 포함) PuzzleVault 프로젝트에서 진행된 오류 수정, 게임 밸런싱, 기능 개선 및 시스템 확장에 관한 모든 상세 작업 기록을 담고 있습니다.

## 1. 개별 게임 기능 고도화 및 치명적 버그 수정

### 🎮 TileTurn (타일 턴)
- **절대 풀 수 없는 퍼즐 문제(Solvability) 완벽 해결**: Cascade(연쇄) 모드에서 배열 역연산 과정의 맹점으로 인해 클리어가 불가능한 맵이 생성되는 치명적 논리 오류를 수식 마스크 계산(Backward Step Tracing) 알고리즘으로 전면 교체. 이제 100% 모든 퍼즐이 수학적인 해답을 가짐.
- **Auto-Solve (자동 시연) 힌트 기능 전면 개편**: 
  - 잦은 힌트 버튼 클릭 대신, 힌트 활성화 시 완성형 정답의 수순을 천천히 재생하며 자동으로 타일을 뒤집어주는 시연 시스템 도입.
  - 사용자가 시연 과정을 눈으로 시각적으로 따라가기 쉽도록 클릭 딜레이 애니메이션 2배 연장 하향.
- **Auto-Solve Complete 모달창 개선**: 퍼즐 자동 완료 시 띄워지는 모달 창의 모든 버튼(Restart Puzzle, View Board, 닫기 ✕) 클릭 시 필수로 구글 애드센스 리워드 광고가 실행되도록 수익화 구조 연동. 광고 시청 후 재시작 버튼은 새 게임 셋업, 닫기와 보드보기 버튼은 정답 뷰를 유지하도록 기능 분리. 배경 클릭을 통한 꼼수 우회 완전 통제.
- **통계(Stats) 창 누락 추가**: 기존에 빠져있던 전체 게임 통계 팝업 모달창 스크립트를 새로 추가 연결.

### 🎮 PipeLink (파이프 링크)
- **맵 생성기(Generator) 오류 디버깅**: Pack 4(듀얼 모드) 등을 포함하여 목적지에 도달 가능한 길이 생성되지 않는 불가능한 스크립트 맵 구조 이슈를 확인하여 항상 클리어 가능한 회로 연결이 생성되도록 알고리즘 재정립.
- **Auto-Solve 모달창 UI 트리거 통일**: TileTurn과 동일하게 오토솔브 모달에서 세 가지 버튼을 통한 광고 트리거 체계를 분리 작성. 배경 우회 차단 및 순수 정답 확인(View Board) 목적과 재도전 패스 분리 적용 완료.

### 🎮 HexMatch (헥스 매치)
- **게임 오버 논리 오류 정밀 수정**: 헥사곤의 첫 번째 열(최상단)이 비어있었다고 해서 게임이 비정상적으로 조기 종료되어버리던 조건을 타당하게 고침.
- **빈칸 채우기(fillEmpty) 및 셔플(Shuffle) 기능 정상화**: 중간에 꼬여버린 타일들에 대한 버그 디버깅 및 해결.
- **수동 셔플/힌트 기능 신규 추가**: 플레이어가 실력으로 풀다가 매칭할 블록이 더 이상 없는 꽉 막힌 데드락 상태가 되었을 때 보드를 임의로 섞거나 힌트를 통해 탈출할 수 있는 UI 버튼 제공.

### 🎮 ColorFlow (컬러 플로우)
- **힌트 UI의 중복 렌더링 에러 해결**: 결과 모달창에서 추천 게임 리스트 및 힌트가 두 번씩 겹쳐 노출되어 디자인이 깨지는 이슈 보완.
- **사용성 평가 후 힌트 기능 삭제**: 해당 게임의 구조상 오히려 힌트가 몰입감을 떨어뜨리는 부작용이 있어 사용자 요청에 따라 ColorFlow에서 힌트 기능을 영구 제거하는 방향으로 확정.

### 🎮 MergeChain (머지 체인)
- **고난이도 피로도 해소 (Freeze 기믹 추가)**: 단조로움을 피하고 체감 난이도를 높이기 위해, 일정 시간 동안 병합되지 못하고 바닥에 방치된 공들이 굳어버려 병합 조건을 불만족하게 만드는 'Freeze'(동결) 메커니즘을 심화 시스템으로 도입.
- **드롭 풀 및 게임 안내 갱신**: 게임 환경이 빡빡해진 만큼 나타날 수 있는 공의 종류를 다각화했으며, 새로 추가된 동결 룰을 사용자가 숙지하도록 "How to Play" 가이드 원문 최신화 적용.

### 🎮 SortStack (소트 스택)
- **게임 프리징 버그 수정**: 게임 도중 힌트 버튼을 누르면 스크립트 충돌로 화면이 일시 정지되거나 반응하지 않던 블랙아웃 현상의 범인 색출 및 동작 무결성 조치.

## 2. 웹사이트 기능 및 전체 게임 시스템 (Global & Infra)

- **공통 사운드 효과 (SFX) 토글 버그 일괄 수정**: SortStack, QuickCalc, TileTurn, ColorFlow 등 여러 게임에서 사운드(스피커 🔊) 음소거 버튼을 눌러도 아이콘 텍스트 자체가 바뀌지 않던 비주얼 버그를 완전히 수정하여 즉각적인 피드백 제공.
- **기본 내비게이션(Footer) 링크 교정**: 사이트 하단에 위치한 공통 약관 및 설명 링크들 (About, Privacy, Terms, Contact)의 클릭 미스매치를 수정하고, 로컬 환경에서 서비스 워커에 의해 스토리지와 라우팅이 꼬이는 문제를 차단 시킴.
- **모달창 범용성 보완**: 모든 게임에서 "닫기 (✕)" 버튼 자체가 HTML에 등록되지 않았던 페이지들을 전체 스캔하여 추가하였으며, 전반적인 모달 사용성을 향상.
- **가시성 개선(파비콘)**: 브라우저 탭 상단에 노출되는 파비콘 아이콘의 여백 및 디자인 사이즈를 재조정하여 웹브라우저에서 게임 식별력을 높였음.
- **자바스크립트 캐싱 에이징 꼼수 대응**: 각종 변경 내역이 로컬 브라우저에서 먹지 않던 문제들을 `v=N` 스크립트 파라미터 업데이트를 통해 상시 캐시 클리어되도록 강제 유지관리.
