## 1. 프로젝트 요약 (Project Overview)
- **도메인 & 목적**: 게임 12종을 브라우저에서 무료로 플레이하고 광고 수익(AdSense)을 창출하는 퍼즐 웹사이트 (https://puzzlevault.pages.dev)
- **개발 스택**: 프레임워크 없는 순수 HTML5, Vanilla JavaScript, CSS (빌드 도구 없음)
- **최종 무결성 달성 여부**: **[100% 바이럴 출시 준비 완료 상태]**
- **아키텍처 규칙**:
  - `css/global.css`: 전체 디자인 시스템 담당 (변수화, 모바일/아이폰 노치 Safe Area 완벽 적용)
  - `js/common.js`: 공통 네비게이션, 크로스프로모 연산, 통계 저장, 힌트 매니저(광고 연동) 담당
  - `js/i18n.js`: 5개 국어(한/영/일/중/스페인) 자동 로드 및 실시간 번역
  - `js/adsense.js`: 3게임 유예 후 전면 광고 송출, 로비 진입 시 60초 쿨다운 준수한 하단 광고 새로고침
  - `sw.js`: 오프라인 PWA 구동 지원 (캐시 전략 최적화 완료, 현재 버전 v11)

## 2. 12종 게임 라인업 완성 상태
초기 10종에 이어, **최상위 IQ 두뇌 테스트용 프리미엄 2종(ShapeIQ, LogicIQ)을 메인 이벤트로 편입**하여 완전히 확장 완료했습니다.
1. `numvault` (숫자 자물쇠 추리)

3. `patternpop` (패턴 암기력)
4. `sortstack` (색상 분류 병합)
5. `quickcalc` (초속 사칙연산)
6. `tileturn` (타일 색상 뒤집기)
7. `colorflow` (헥사 라인 연결)
8. `pipelink` (회로 연결)
9. `mergechain` (숫자 연속 드롭다운)
10. `hexmatch` (6방향 같은색 매치)
11. **`shapeiq` (도형 시각 논리 분석) — NEW (디버그 완벽 적용)**
12. **`logiciq` (작업 기억 코르시 블록) — NEW (UI/동작 스탠다드 적용)**

## 3. 오늘(2026.03.26) 진행한 크리티컬 패치 이력
이 기록을 통해, 어떠한 오류가 수정되었는지 "기억"하십시오.
1. **Sitemap 누락 대응**: 신규 IQ 게임 2종을 `sitemap.xml`에 priority 0.9로 배치 완료하여 구글 크롤링 불가 현상 해결.
2. **ShapeIQ 논리 오류 수정**: 시퀀스 룰 랜덤 제너레이션 시 모형과 색깔이 완전히 동일하게 나오는 중복 맹점을 루프로 방어. FAQ에 서술된 '데일리 모드 3분 오버' 종료 트리거(`elapsed >= 180`) 적용 완료.
3. **LogicIQ UI/UX 개선**: 이미 탭했던 블록이 고정 디밍(dimming)되어 다음 라운드의 똑같은 블록 선택 시 혼동을 주던 치명적 UX 버그 완전 제거.
4. **QuickCalc 공유 기능 크래시 패치**: 
   - [전역 감사(Global Audit) 결과 발각된 버그] 결과화면(Result Modal)에서 `Share`를 눌렀을 때, 로직 내부에서 존재하지 않는 구버전 전역객체 `G`를 참조하여 `ReferenceError`가 발생하고 작동이 중지되는 치명적 버그를 발견.
   - `quickcalc-logic.js` 내부의 해당 구문을 현재 쓰는 `state.score` 와 템플릿 리터럴로 전체 교체 완료하여 백신 처리함.
