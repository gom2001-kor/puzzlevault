How to Play NumVault

NumVault challenges you to crack a secret number code through logical deduction. Each puzzle presents a hidden code of 4 to 6 digits, and your goal is to figure it out within a limited number of attempts.

After each guess, you receive color-coded feedback for every digit. A green indicator (🟢) means the digit is correct and in the right position. Yellow (🟡) means the digit exists in the code but is in the wrong position. Gray (⚫) means the digit is not in the code at all.

Use the Number Tracker — a unique NumVault feature — to keep track of what you know about each digit (0–9). The tracker automatically updates after each guess, showing whether a digit is confirmed, included, excluded, or still unknown.

NumVault offers three exciting modes. Daily Challenge gives everyone the same puzzle each day — compete with friends and track your streak. Free Play lets you choose any difficulty from Easy (4 digits, 8 attempts) to Expert (6 digits, 10 attempts). Speed Mode is a 60-second race to solve as many codes as possible, earning bonus time for each correct answer.

Strategic tip: start with digits that cover a wide range (e.g., 1234, then 5678) to quickly narrow down which digits are in the code. Then use the yellow and green feedback to place them correctly. The Number Tracker is your best friend — use it to eliminate possibilities systematically.

Frequently Asked Questions

What makes NumVault different from other number guessing games?

NumVault features a unique Number Tracker that visually tracks the state of each digit (0–9) as you play. It also offers Speed Mode where you race against a timer, and multiple difficulty levels including Expert mode with 6-digit codes.

How does the feedback system work?

After each guess, every digit gets a color: 🟢 Green means correct digit in the correct position. 🟡 Yellow means the digit exists in the code but in a different position. ⚫ Gray means the digit is not in the code. Green matches are assigned first, then yellow for remaining unmatched digits.

Can the code contain duplicate digits?

It depends on the difficulty level. Easy mode has no duplicate digits, but Normal, Hard, and Expert modes allow duplicates, adding an extra layer of challenge to your deduction.

When does the Daily Challenge reset?

The Daily Challenge resets at midnight UTC every day. Everyone around the world gets the same puzzle, so you can compare results with friends. Your streak tracks consecutive days of successful completion.

How do hints work?

Hints reveal the correct digit and position for one unconfirmed slot. In Daily mode you get 1 hint maximum, and in Free Play you get up to 2. Using a hint is noted in your share results with a 💡 symbol and applies a score penalty.
