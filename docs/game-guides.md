## 🔢 Game 1: NumVault — 숫자 추론 퍼즐

### 개요
비밀 숫자 코드를 추리하는 게임입니다. 숫자를 입력하면 색깔 피드백으로 힌트를 주고, 제한된 횟수 안에 정답을 맞추면 성공!

### 게임 진행
1. 게임이 시작되면 **비밀 코드** (4~6자리 숫자)가 숨겨집니다
2. 넘패드로 숫자를 입력하고 **Enter**를 눌러 추측을 제출합니다
3. 각 자리에 **색깔 피드백**이 표시됩니다:

| 색깔 | 의미 |
|------|------|
| 🟢 **초록** | 숫자도 맞고 위치도 정확 |
| 🟡 **노랑** | 숫자는 맞지만 위치가 다름 |
| ⚫ **회색** | 코드에 없는 숫자 |

4. 피드백을 분석하여 다음 추측을 더 정확하게 합니다
5. **제한 횟수** 안에 정답을 맞추면 성공!

### Number Tracker (핵심 기능)
그리드 아래에 **0~9** 숫자 트래커가 있습니다:
- **?** → 아직 모르는 상태
- **✓** (초록) → 정답에 있고 위치도 확인됨
- **~** (노랑) → 정답에 있지만 위치 미확인
- **✕** (회색) → 정답에 없는 숫자

자동으로 업데이트되므로 어떤 숫자가 남았는지 한눈에 파악 가능!

### 모드
| 모드 | 설명 |
|------|------|
| **Daily** | 매일 전 세계 동일한 퍼즐, 4자리, 6회 시도 |
| **Free Play** | 난이도 선택 가능 (Easy~Expert) |
| **Speed** | 60초 안에 최대한 많이 풀기! 정답 시 +15초 |

### 난이도 (Free Play)
| 난이도 | 자릿수 | 시도 횟수 | 중복 숫자 |
|--------|--------|-----------|----------|
| Easy | 4자리 | 8회 | ❌ |
| Normal | 4자리 | 6회 | ✅ |
| Hard | 5자리 | 8회 | ✅ |
| Expert | 6자리 | 10회 | ✅ |

### 💡 힌트 버튼
넘패드의 전구(💡) 버튼을 누르면 **아직 못 맞춘 위치 중 하나**의 정답을 알려줍니다.
- Daily: 1회 / Free Play: 2회 / Speed: 사용 불가
- 힌트 사용 시 점수에 50점 패널티

### 공략 팁
1. 첫 추측은 **겹치지 않는 숫자**로 (예: 1234 → 5678)
2. 🟢가 나오면 그 자리는 고정, 나머지 자리를 탐색
3. 🟡이 나오면 그 숫자를 **다른 위치**에 넣어보기
4. ⚫인 숫자는 **완전 제거** — Number Tracker 확인!

## 🧱 Game 2: GridSmash — 블록 배치 퍼즐

### 개요
10×10 그리드에 블록 조각을 배치하여 줄을 완성하고 클리어하는 퍼즐입니다. 더 이상 조각을 놓을 수 없으면 게임 오버!

### 게임 진행
1. 화면 아래 **피스 독(dock)**에 **3개의 블록 조각**이 주어집니다
2. 조각을 **드래그**하여 10×10 그리드 위에 놓습니다
3. 가로 또는 세로 한 줄이 모두 채워지면 → **줄 클리어!** ✨
4. 3개를 모두 배치하면 새 조각 3개가 나옵니다
5. 새 조각 3개 중 **하나도 놓을 수 없으면** → GAME OVER 💥

### 드래그 & 드롭 조작
- **PC**: 마우스로 조각을 클릭하고 드래그
- **모바일**: 손가락으로 터치하고 드래그
- 초록색 프리뷰 = 놓을 수 있음 ✅
- 빨간색 프리뷰 = 놓을 수 없음 ❌
- 그리드 밖에서 손을 떼면 조각이 독으로 돌아감

### 🔄 블록 회전 (Block Rotation)
블록 조각을 **90° 시계방향**으로 회전하여 원하는 방향으로 맞출 수 있습니다!
배치 전이나 배치 도중 언제든 회전이 가능합니다.

#### 📱 모바일 / 터치스크린 조작법
1. 화면 아래 **피스 독(dock)**에 있는 블록 조각을 **가볍게 탭**합니다 (드래그하지 마세요!)
2. 탭할 때마다 블록이 **90° 시계방향**으로 회전합니다
3. 원하는 방향이 될 때까지 **여러 번 탭** → 1번 탭 = 90°, 2번 = 180°, 3번 = 270°, 4번 = 원래 방향
4. 방향이 결정되면 **길게 누른 채 드래그**하여 그리드에 배치합니다

> 💡 **핵심 포인트**: 짧게 탭 = 회전, 길게 누르고 이동 = 드래그. 이 차이를 기억하세요!

#### 🖥️ PC / 데스크톱 조작법
1. 블록 조각을 **왼쪽 클릭(좌클릭)**으로 잡고 드래그를 시작합니다
2. 드래그하는 도중에 **오른쪽 클릭(우클릭)**을 누르면 블록이 **90° 시계방향** 회전합니다
3. 그리드 위에 놓기 전까지 **여러 번 우클릭**하여 원하는 방향으로 맞출 수 있습니다
4. 회전이 완료되면 원하는 위치에 **마우스 버튼을 놓아** 배치합니다

> 💡 **핵심 포인트**: 좌클릭 유지 + 우클릭 = 드래그 중 회전. 프리뷰가 즉시 반영됩니다!

#### 회전 동작 원리
| 탭/클릭 횟수 | 회전 각도 | 설명 |
|-------------|----------|------|
| 1번 | 90° | 시계방향 1/4 회전 |
| 2번 | 180° | 상하좌우 반전 |
| 3번 | 270° | 반시계방향 1/4 회전과 동일 |
| 4번 | 360° | 원래 방향으로 복귀 |

#### 회전 활용 전략
1. **좁은 공간에 끼워넣기** — L자나 T자 조각을 회전하면 빈틈에 딱 맞출 수 있습니다
2. **줄 클리어 완성** — 가로줄이 거의 찼는데 블록 방향이 안 맞으면? 회전으로 해결!
3. **미리 회전해두기** (모바일) — 독에서 먼저 방향을 맞춘 후 드래그하면 실수 줄임
4. **Shatter Zone 공략** — 하단 2줄에 블록을 정확히 넣으려면 회전이 필수!

### 줄 클리어 규칙
- **가로줄** 10칸 또는 **세로줄** 10칸이 모두 차면 클리어
- 여러 줄을 동시에 클리어하면 **콤보!** (점수 배율 증가)
- 연속 턴 클리어 시 **스트릭** 보너스

### ⭐ 특수 블록 (Turn 16부터, 20% 확률)
| 블록 | 외형 | 효과 |
|------|------|------|
| 💎 **Crystal** | 반짝이는 효과 | 놓는 순간 3×3 범위 클리어 |
| 🧊 **Frost** | 하얀 테두리 | 빈 셀 2개를 얼림 (2번 클리어해야 제거) |
| ⭐ **Wild** | 금색 1×1 | 별도 슬롯, 빈 칸 아무 곳에 배치 가능 |

### 🔥 Shatter Zone (10턴마다)
- 하단 2줄이 **금색**으로 빛남
- 이 영역에서 줄 클리어 시 **점수 3배!**
- 5턴 지속 — 만료 시 빈 칸 50%가 랜덤 블록으로 채워짐
- 타이밍을 노려서 콤보를 터트리세요!

### 모드
| 모드 | 설명 |
|------|------|
| **Classic** | 끝없는 모드, 게임 오버까지 |
| **Daily** | 시드 기반 조각 순서, 3분 제한시간 |
| **Zen** | 점수/Shatter Zone 없이 편안하게 |
| **Sprint 40** | 40줄 클리어 — 시간 기록 도전 |

### 공략 팁
1. **가장자리부터** 채우면 중앙에 공간 확보 가능
2. **큰 조각 먼저** 배치하고 작은 조각으로 마무리
3. 한 줄이 거의 다 찼으면 **우선 클리어** — 공간이 핵심!
4. Shatter Zone이 나타나면 **하단에 집중**하여 3배 점수 획득

## 🎯 Game 3: PatternPop — 패턴 기억 게임

### 개요
화면에 잠깐 나타나는 파란색 셀들의 위치를 기억한 뒤, 정확히 탭하는 기억력 게임입니다.

### 게임 진행 (3단계 반복)

#### 1단계: 👀 MEMORIZE (암기)
- 그리드 위에 **진한 파란색 셀**들이 켜집니다
- 이 위치를 **눈으로 기억**하세요!
- 약 2.5초 후 사라짐 (라운드가 높을수록 짧아짐, 최소 1.2초)

#### 2단계: 🎯 RECALL (맞추기)
- 그리드가 빈 흰색으로 돌아갑니다
- **기억한 위치를 탭/클릭**하세요
- ✅ 정답 → 초록색 + ✓
- ❌ 오답 → 빨간색 + ✕ → **하트 -1**
- 모든 정답을 다 탭하면 라운드 성공!

#### 3단계: 📊 FEEDBACK (결과)
- 맞춘 셀/틀린 셀/못 찾은 셀이 색으로 표시됩니다
- 점수가 추가된 후 다음 라운드로 진행

### 난이도 증가
| 라운드 | 그리드 | 기억할 셀 |
|--------|--------|-----------|
| 1~4 | 3×3 (9칸) | 3~5개 |
| 5~8 | 4×4 (16칸) | 5~8개 |
| 9~14 | 5×5 (25칸) | 9~13개 |
| 15+ | 6×6 (36칸) | 14개+ |

### ⚡ 디코이 플래시 (Round 10부터!)
PatternPop의 **핵심 트릭**입니다:
- 암기 단계에서 진짜 패턴 외에 **연한 하늘색 셀**이 **0.3초만** 반짝합니다
- 이것은 **가짜(디코이)**입니다!
- RECALL 때 디코이 위치를 탭하면 → 오답 처리 (하트 -1)

**구별법:**
| 구분 | 진짜 타겟 | 디코이(가짜) |
|------|----------|------------|
| 색깔 | **진한 파란색** | 연한 하늘색 |
| 지속시간 | **오래** 켜져 있음 | **0.3초**만 깜빡 |

### ❤️ 라이프
- 시작 시 **하트 3개**
- 오답 탭마다 **1개 감소**
- 0개 = **GAME OVER** 💥

### 점수
- 정답 1개: 10 × 라운드 보너스
- 퍼펙트 보너스: 한 라운드 무실점 클리어 시 추가 점수
- 스피드 보너스: 5초 이내 빠른 응답 시 추가 점수

### 모드
| 모드 | 설명 |
|------|------|
| **Endless** | 끝없이 도전, 라이프 다하면 종료 |
| **Daily** | 매일 동일 패턴, 점수 비교 가능 |

### 공략 팁
1. 개별 위치보다 **모양/패턴으로 기억** (L자, 대각선, ㄱ자 등)
2. 디코이는 **짧고 연하게** 깜빡이니 오래 켜진 것만 기억
3. 확실한 것부터 탭하고, 애매한 것은 건너뛰기
4. 그리드가 커지면 **영역별로 나눠서** 기억 (왼쪽 위, 오른쪽 아래 등)

## 📚 Game 4: SortStack — 서랍 정렬 혼돈!

### 목표
뒤죽박죽 섞여 있는 색깔 블록들을 **한 서랍에 같은 색상의 블록 4개씩** 예쁘게 모으는 게임입니다.

### 1. 기본 조작법 (블록 옮기기)
- **들기:** 아무 서랍이나 터치(클릭)하면 서랍 맨 위에 있는 블록이 살짝 위로 붕~ 뜹니다. 잡았다는 뜻입니다!
- **놓기:** 그 상태에서 넣고 싶은 다른 서랍을 터치하면 블록이 휙- 날아갑니다. (만약 다른 서랍에 두기 싫어 취소하고 싶다면, 방금 터치했던 원래 서랍을 한 번 더 누르면 원래대로 쏙 들어갑니다.)
- **규칙:** 블록은 아무 데나 들어가지 않습니다! **[아예 텅 빈 서랍]**이거나, **[그 서랍 맨 위에 있는 블록과 색상이 완전히 똑같을 때]**만 조립이 가능합니다. 그리고 한 서랍에는 최대 4개의 블록까지만 들어갑니다.

### 2. 숨겨진 쾌감 보너스! (✨Lock & Expand)
이 게임만의 최고 꿀잼 요소입니다.
고생해서 한 서랍을 완전히 똑같은 색상 4개로 꽉 채우면, 반짝! 파티클이 터지며 그 서랍이 **🔒 자물쇠** 모양과 함께 완벽히 잠겨서 완성 처리가 됩니다.
그리고 아주 기분 좋은 보상으로... **맨 끝부분에 빈 서랍이 하나 더 추가(!)** 됩니다. 색을 완성할수록 공간이 넓어지니 후반부 여유가 엄청나게 커집니다.

### 3. "아차 실수!" 완전 무료 되돌리기 (↩️ Undo)
잘못 옮겨서 꽉 막혔나요? 스트레스받지 마세요.
상단의 **↩️ Undo** 버튼을 누르시면 조건 없이, 횟수 제한 없이, 광고 볼 필요 없이 이전 상태로 즉각 되돌려 줍니다. 10번 전으로 돌리고 싶으면 10번 연타하셔도 됩니다! 언제든 완벽한 구상을 다시 세우세요.

### 4. 하드코어 플레이어 주의사항 (⚠️ 횟수 제한)
**[Easy(색 4개)]** 모드와 **[Medium(색 6개)]** 모드는 제한 횟수 없이 마음껏 옮기며 느긋하게 힐링할 수 있습니다. 
하지만 두뇌를 시험하는 **[Hard(색 8개)]**와 **[Expert(색 10개)]** 모드를 선택하면 화면 상단에 째깍째깍 **남은 이동 횟수 (예: 40번)**가 표시됩니다. 쓸데없는 이동으로 횟수를 전부 소진하면 그대로 게임 오버가 되니 신중한 콤보를 설계해야 합니다.



## 📚 Game 5: QuickCalc — 스피드 수학 아케이드!

### 목표
QuickCalc는 빠른 계산 능력과 반사신경을 시험하는 아케이드 형식의 수학 퍼즐 게임입니다. 주어지는 30초의 시간 안에 최대한 많은 공식을 풀어내세요!

### 1. ⏱️ 시간 관리와 점수 (Time & Score)
- **시작:** 게임은 기본 시간 **30.0초**로 시작됩니다.
- **정답 시 (✅):** 시간이 **+2초** 추가되며 점수를 획득합니다!
- **오답 시 (❌):** 찰나의 실수에 자비는 없습니다. 시간이 가차없이 **-3초** 차감되며 화면이 크게 진동합니다.
- 상단 바의 시간이 0초 구역에 닿아 타이머가 멈추면 게임이 끝납니다!

### 2. 🔥 콤보 보너스 시스템
연속해서 정답을 고를수록 시간 보너스가 더욱 강화됩니다.
- **3연속 정답 시: 1문제당 +3초** 추가
- **5연속 정답 시: 1문제당 +4초** 추가
- **10연속 정답 시: 1문제당 +5초** 추가
- 한 번이라도 틀리면 콤보는 가차 없이 0부터 다시 시작하므로, 속도만큼이나 정확도가 중요합니다!

### 3. 🎯 스테이지 & 연산자 룰렛 (Operator Roulette)
정답을 많이 맞춰 나갈수록 수식의 형태와 숫자의 크기가 점점 더 똑똑하고 복잡하게(자동 난이도 스케일링) 발전합니다. 
특히 15번 문제를 통과하면 **'연산자 룰렛(Operator Roulette)'** 모드가 새롭게 등장합니다. 숫자가 아닌 **연산자 기호 (+, -, ×, ÷)**를 맞혀야 하는 거꾸로 생각하기 모드에도 대비하세요!

### 4. 자주 묻는 질문 (FAQ)
- **타이머 시스템의 최대 한도 시간이 있나요?**
  아뇨, 한도 시간 캡은 없습니다. 연속해서 콤보를 달성하고 초과 시간을 끊임없이 적립하면 30초에서 시작했음에도 실제로는 1분이 넘어가는 시간을 벌 수도 있습니다.
- **오답 보기에 규칙이 있나요?**
  네! 단순 무작위가 아닙니다. 게임 속 정답 생성기는 플레이어를 헷갈리게 만들기 위해 **정답과 1~3 정도만 차이 나는 값 1개**, **10 언저리가 차이 나는 값 1개**, 그리고 무작위 값 1개를 철저하게 계산하여 4지 선다로 제공합니다.
- **Daily(일일) 모드와 Free Play의 차이는 무엇인가요?**
  Daily 모드는 무작위 난수가 전 세계 모든 사람에게 일별로 동일하게 고정 제공됩니다(Seed 사용). 즉, 매일 같은 순서의, 똑같은 수식이 제공되어 친구와 공정하게 경쟁할 수 있습니다. Free Play는 들어갈 때마다 완전한 무작위로 계속 제공됩니다.

## 🔄 Game 6: TileTurn — 타일 뒤집기 퍼즐

### 목표
TileTurn은 고전적인 'Lights Out' 퍼즐을 현대적으로 재해석하여 연쇄 반응을 일으키는 전략 퍼즐입니다. 타일을 뒤집어 화면상의 모든 타일을 켜지게 만들거나, 한 가지 색상으로 통일하세요.

### 1. 🎮 3가지 다른 플레이 모드
게임에는 총 7개의 팩이 존재하며, 각 팩은 다음 3가지 모드 중 하나의 규칙을 따릅니다.
- **Classic (클래식) 모드:** 화면의 모든 타일을 밝게(ON) 불이 들어온 상태로 만들어야 합니다. 어떤 타일을 탭하면 그 타일과 바로 위, 아래, 왼쪽, 오른쪽(직자 모양)의 상태가 반전(ON/OFF)됩니다.
- **Cascade (캐스케이드) 모드:** 기본적인 방식은 클래식 모드와 동일하지만 파급력이 파도처럼 2단계로 퍼집니다. 한 타일을 탭하면 주변 4칸뿐만 아니라 그 옆의 타일들(총 최대 13칸)이 연쇄적으로 뒤집힙니다!
- **Spectrum (스펙트럼) 모드:** 타일이 단순히 켜지고 꺼지는 것이 아니라 3가지 예쁜 색상(Coral → Amber → Blue)을 차례대로 순환합니다. 탭하면 타일과 인접 4칸의 색상이 다음 색으로 일제히 변경됩니다. 모양과 상관없이 보드 전체가 **단 하나만의 동일한 색상**으로 통일되면 클리어입니다!

### 2. ⭐ 완벽주의자를 위한 별점 시스템 (Star Rating)
모든 레벨은 언제나 가장 빠르고 완벽하게 풀 수 있는 **최소 이동 횟수(Minimum Moves)**가 내부적으로 정해져 있습니다.
- ⭐⭐⭐ (3개): 지정된 최소 이동 횟수 안에 퍼즐을 해결했을 때 획득! 
- ⭐⭐ (2개): 아쉽게도 최소 이동 횟수보다 1~2번 더 많이 탭해서 깼을 때.
- ⭐ (1개): 횟수를 많이 낭비했지만, 어찌저찌 풀어서 클리어는 해냈을 때.

### 3. 무제한 되돌리기 (Undo) ↩️
"아, 한 번만 뒤로 돌아가고 싶은데..." 하실 필요가 전혀 없습니다!
화면 아래쪽에 있는 `Undo(되돌리기)` 버튼을 누르면 이전에 탭하기 전 상태로 보드가 즉시 돌아갑니다. 동시에 **가장 중요한 점: 올라갔던 이동 횟수 카운터도 다시 깎아줍니다!** 
따라서 별 3개를 노리기 위해 이 버튼을 무제한으로 공짜로, 어떤 페널티 없이 마음껏 활용하시면서 여러 패턴을 자유롭게 실험해 보세요.

### 4. 자주 묻는 질문 (FAQ)
- **혹시 아예 깰 수 없는 퍼즐이 나오기도 하나요?**
  절대 불가능합니다. TileTurn의 모든 레벨은 아예 '완성된 깔끔한 상태'에서 컴퓨터가 플레이어 몰래 일정 횟수 위장 탭을 가하여 흩어놓는 **역추적 방식(Reverse Method)**으로 생성됩니다. 이미 답이 존재했던 상태에서 섞었기 때문에 100% 클리어가 가능합니다.
- **Spectrum 모드는 어떤 색으로 맞춰야 하나요?**
  3가지 색상 중 어느 색깔이든 상관없습니다. 모든 칸이 Coral이어도 되고, 전부 Blue이거나 모두 Amber여도 똑같이 클리어로 인정됩니다. 화면 위쪽의 💡전구 마크가 빛나면 통일된 것입니다.

## 🎨 Game 7: ColorFlow — 색상 경로 연결 퍼즐

### 목표
ColorFlow는 같은 색상의 두 다이아몬드(◆) 수정들을 겹치지 않는 에너지 경로로 연결하는 논리 퍼즐입니다. 보드의 빈 공간을 하나도 남기지 않고 100% 꽉 채워 모든 색상을 연결하세요.

### 1. 기본 조작과 덮어쓰기 (Overwrite/Sever)
- **그리기:** 맵에 놓인 한 다이아몬드를 터치한 채로 인접한 빈칸으로 드래그하면 그 색상의 두꺼운 선이 그려집니다. 같은 색상의 반대편 다이아몬드에 정확히 맞닿으면 연결(Connected) 처리되며 반짝이기 시작합니다.
- **끊기 (Overwrite):** 선들은 겹칠 수 없습니다. 선을 그리다가 이미 칠해진 "다른 색깔"의 선을 파고들면, 내가 그은 새 선이 덮어씌워지며 기존에 있던 상대 선은 그 부분부터 끝부분까지 전부 삭제(단절)됩니다.
- **지우기 (Eraser):** 내가 그려놓은 전체 선을 한 번에 날려버리고 싶다면, 해당 선의 출발점이나 도착점 다이아몬드를 **아무 곳이나 한 번 톡! 탭(터치)** 하세요. 그 색상의 경로만 깨끗하게 증발합니다. 선을 따라 거꾸로 되돌아가면 부분적으로 지우는 것도 가능합니다.

### 2. 💯 Coverage (커버리지 점수)와 별점
그저 시작과 끝 다이아몬드를 잇기만 하면 끝나는 게임이 아닙니다!
- 진정한 퍼즐의 완성은 화면상의 **단 한 칸의 빈 칸(하얀 칸)도 없이 모든 칸을 특정 색상의 경로로 채워 넣어야** 달성됩니다.
- 게임 중 실시간으로 상단에 `Coverage: X%` 로 내가 공간을 얼마나 낭비 없이 쓰고 있는지 표시됩니다. 
- 모든 퍼즐을 다 이었고 커버리지가 **100%** 라면 완벽한 클리어로 인정되어 ⭐⭐⭐ 별 3개를 즉시 획득합니다!

### 3. 고인물을 위한 `Flow Bonus (플로우 효율 점수)`
- 결괏값 화면에 보이는 추가 점수입니다! 수학적으로 목표 지점까지 갈 수 있는 "직선상의 절대 최단 거리(Manhattan Distance)"가 존재하죠.
- 만약 플레이어가 100% 커버리지를 채우기 위해 맵을 이리저리 빙빙 돌아가지 않고, 그 절대 최단 거리의 1.5배 이내로 가장 완벽하게 직선에 가깝게 선을 그으면서도 보드를 다 채우는 천재적인 루트를 발견했다면 해당 선마다 **+50점의 보너스 점수**를 추가로 드립니다. 

### 4. 자주 묻는 질문 (FAQ)
- **Undo Path 버튼을 여러 번 누를 수 있나요?**
  네, 무제한으로 누를 수 있습니다! ColorFlow의 `Undo Path` 버튼은 단순히 전체 맵을 지우는 것이 아니라, 여러분이 직전에 했던 행동(선 그리기, 선 지우기, 심지어 전체 초기화까지) 단위의 과거 상태를 완벽하게 기억해서 하나씩 역순으로 되돌려 줍니다. 완전히 판을 엎고 백지상태부터 다시 하고 싶을 땐 `Reset All` 기능을 사용하세요.
