### 로컬 개발 시 주의사항
- SW가 캐싱으로 인해 변경 사항이 반영되지 않을 수 있음
- `navigator.serviceWorker.register('/sw.js')`는 절대 경로이므로 로컬 서버(Live Server 등) 사용 권장
- 변경 후 브라우저 DevTools → Application → Service Workers → Unregister 후 새로고침
