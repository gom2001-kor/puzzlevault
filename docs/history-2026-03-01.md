## 향후 참고 사항

### 블로그 포스트 추가 방법
1. `/blog/posts/_template.html`을 복사하여 `/blog/posts/[slug].html` 생성
2. `<article class="blog-content">` 안에 포스트 본문 작성
3. `js/blog-data.js`의 `BLOG_POSTS` 배열에 메타데이터 추가
4. `sitemap.xml`에 새 URL 추가
→ 블로그 목록 페이지는 자동 업데이트됨

### 로컬 개발 시 주의사항
- SW가 캐싱으로 인해 변경 사항이 반영되지 않을 수 있음
- `navigator.serviceWorker.register('/sw.js')`는 절대 경로이므로 로컬 서버(Live Server 등) 사용 권장
- 변경 후 브라우저 DevTools → Application → Service Workers → Unregister 후 새로고침
